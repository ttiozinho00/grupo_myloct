
# Rodar o codigo

- Instalar [MongoDB](https://www.mongodb.com/try/download)
- entre na pasta backend
```bash
cd server
```
- instalar todos os pacotes
```bash
npm install
```
- rode o backend
```bash
npm start
```
o serve esta rodando http://localhost:5000/

- entre na pasta client
```bash
cd client
```
- instalar todos os pacotes
```bash
npm install
```
- rode  a aplicação
```bash
npm start
```
O app está rodando  http://localhost:3000/
